# Finanzas

## Para que sirve
Registra y consulta ingresos, egresos, deudas, cobros, pagos, perdidas y reembolsos.

## Que se genera automatico
- Ventas completadas
- Compras recibidas
- Produccion completada
- Mermas

## Crear un movimiento manual
1. Abre `Finanzas`.
2. Pulsa `Nuevo movimiento`.
3. Selecciona tipo, concepto, monto y fecha.
4. Guarda.

## Balance
El balance considera ingresos menos egresos que afecten saldo.
