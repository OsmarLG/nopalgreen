# Ordenes de produccion

## Para que sirve
Ejecuta una receta y registra produccion real.

## Flujo basico
1. Abre `Ordenes de produccion`.
2. Crea una nueva orden.
3. Selecciona producto y receta.
4. Define cantidad planeada.
5. Al completar, valida consumos y salida final.

## Importante
Al completar la orden se generan movimientos de inventario y finanzas.
