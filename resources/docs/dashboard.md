# Dashboard

## Para que sirve
El dashboard muestra un resumen rapido del dia segun el rol del usuario.

## Como usarlo
- Entra al sistema y abre `Dashboard`.
- Revisa las tarjetas superiores.
- Usa las listas para ver pendientes, actividad y seguimiento.

## Que cambia por rol
- `admin` y `master`: vista general de operacion.
- `repartidor`: solo entregas propias.
- `planta`: produccion y ordenes.
- `empleado`: asistencia personal.

## Recomendacion
Usa el dashboard como punto de arranque, no como modulo de captura.
