# Materias primas

## Para que sirve
Define los insumos base que entran a compra, inventario y produccion.

## Crear una materia prima
1. Abre `Materias primas`.
2. Captura nombre y unidad base.
3. Asigna proveedor si aplica.
4. Guarda.

## Recomendacion
Usa nombres consistentes para evitar duplicados, por ejemplo `Maiz blanco` y no varias variantes.
