# Reportes

## Para que sirve
Centraliza informacion de asistencia, ventas, compras, produccion, inventario y finanzas.

## Flujo basico
1. Abre `Reportes`.
2. Selecciona `Desde` y `Hasta`.
3. Pulsa `Aplicar`.
4. Revisa resumen, detalle por modulo y desgloses por entidad.

## Exportacion
- `Excel`: descarga un archivo compatible con Excel.
- `PDF`: abre una vista imprimible para guardar como PDF.

## Desgloses disponibles
- Empleado
- Repartidor
- Cliente
- Producto
