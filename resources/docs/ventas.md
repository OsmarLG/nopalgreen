# Ventas

## Tipos de venta
- `Directa`: se vende en el momento.
- `Por entrega`: se asigna a un repartidor y se liquida despues.

## Crear una venta
1. Abre `Ventas`.
2. Pulsa `Nueva venta`.
3. Elige cliente, tipo y estado.
4. Agrega productos.
5. Ajusta precio final si hace falta.
6. Guarda.

## Descuentos
- Si el precio final queda por debajo del precio catalogo, el sistema guarda descuento.
- Debes capturar nota de descuento.

## Por entrega
- Asigna repartidor.
- Registra fecha de entrega.
- Al liquidar, captura vendido y devuelto.
