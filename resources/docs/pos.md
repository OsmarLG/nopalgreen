# POS

## Para que sirve
Captura ventas rapidas sin entrar al modulo completo de ventas.

## Flujo basico
1. Abre `POS`.
2. Crea una tab nueva si necesitas otra venta.
3. Selecciona cliente, tipo de venta y productos.
4. Ajusta cantidades y precio final si aplica.
5. Guarda la venta.

## Tabs pendientes
- Cada tab puede quedarse pendiente.
- Puedes abrir otra venta sin perder la anterior.
- El POS conserva las tabs locales del navegador.

## Nota
El historico completo siempre se revisa en `Ventas`.
