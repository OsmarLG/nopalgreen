# Compras

## Para que sirve
Registra compras de materias primas y productos externos.

## Flujo basico
1. Abre `Compras`.
2. Pulsa `Nueva compra`.
3. Selecciona proveedor.
4. Agrega items con presentacion, cantidad y costo.
5. Guarda.

## Estados
- `Borrador`
- `Recibida`
- `Cancelada`

## Importante
Cuando una compra pasa a `Recibida`, genera movimientos de inventario y finanzas.
