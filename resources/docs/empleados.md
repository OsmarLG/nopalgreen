# Empleados

## Para que sirve
Muestra asistencia, retardos, faltas, faltas equivalentes y dispositivos.

## Como usarlo
1. Abre `Empleados`.
2. Busca por nombre o usuario.
3. Revisa estatus del dia.
4. Entra al detalle si necesitas historial.

## Indicadores
- Asistencias
- Retardos
- Faltas
- Dispositivos registrados
