# Productos

## Para que sirve
Define productos terminados o intermedios para venta, inventario y produccion.

## Datos clave
- Nombre
- Unidad base
- Tipo
- Origen
- Precio de venta

## Precio de venta
El precio catalogo sirve como referencia.
En ventas puede cambiarse, pero el sistema guarda descuento si se vende por debajo.
