# Presentaciones

## Para que sirve
Relaciona una materia prima o producto con su forma de compra, manejo o venta.

## Ejemplos
- Costal 25 kg
- Bolsa 1 kg
- Paquete 500 g

## Uso
1. Abre `Presentaciones`.
2. Elige si es para producto o materia prima.
3. Captura nombre, cantidad y unidad.
4. Guarda.
