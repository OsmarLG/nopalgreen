# Clientes

## Para que sirve
Guarda los datos basicos de los clientes para venta directa o por entrega.

## Crear un cliente
1. Abre `Clientes`.
2. Pulsa `Nuevo cliente`.
3. Captura nombre y datos disponibles.
4. Guarda.

## Editar
- Usa `Editar` para corregir telefono, direccion o estado.

## Recomendacion
Si un cliente ya no se usa, mejor desactivalo en lugar de borrarlo cuando tenga historial.
