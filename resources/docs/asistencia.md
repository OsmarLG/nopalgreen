# Asistencia

## Para que sirve
Configura el checador y el horario laboral.

## Parametros clave
- Hora de entrada
- Hora de salida
- Tolerancia
- Hora a partir de la cual ya es falta
- Cuantos retardos equivalen a una falta
- Dias laborales

## Marcado del empleado
- El empleado ve `Mi asistencia`.
- El sistema muestra codigo diario de entrada y salida.
- Debe capturar el codigo correcto para marcar.

## Dispositivos
Cada marca guarda el dispositivo detectado para auditoria.
